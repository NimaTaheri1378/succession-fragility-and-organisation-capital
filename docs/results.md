# Results

Results are generated by the reproducible pipeline after WRDS extraction, timing checks, and visual QA. Public artifacts are aggregate only.

## Scaled Panel

- Window: 1995-2024 monthly formation dates.
- Firm-month rows: 1,245,777.
- PERMNOs: 7,918.
- GVKEYs: 8,088.
- WRDS extraction: 189 completed shards in the private run log; aggregate panel artifacts are tracked here.

## Baseline OCF

The equal-weight OCF quintile long-short portfolio earns 1.67% annualized excess return with 15.02% annualized volatility and a Sharpe ratio of 0.11. The FF5+UMD alpha is 2.49% annualized with Newey-West t-stat 0.83. The Fama-MacBeth OCF coefficient is small and statistically weak in the current scalar specification.

## Mechanism and Timing

The BoardEx leadership-change event study links 365,226 role-change events to CRSP daily returns. The SEC timing audit was run on the 500 highest-coverage CIKs: 126,891 events have CIK links, 104,726 have a nearby SEC 8-K/proxy filing within the audit window, and there were no SEC fetch errors.

## Model Ladder

The walk-forward model ladder includes Elastic Net, LightGBM, a GPU Deep Sets component proxy, a GPU conditional-autoencoder/SDF block, and a real variable-sized executive-set Deep Sets model built from BoardEx person-role tensors. The leakage-safe executive-set run removes forward exit information, computes prior-employer breadth only from roles started before the formation month, and fits a separate expanding target-year model for each test year from 2018 through 2024.

The timing-safe executive-set result is not sharp enough to support a return-strategy claim. Across 274,998 out-of-sample firm-months, mean rank IC is 0.0079 with a 0.38 t-stat. The score-sorted long-short portfolio earns 0.44% annualized excess return with 10.88% annualized volatility, Sharpe 0.04, and max drawdown -36.78%. The 2021 target year is materially negative, so the original high-Sharpe executive-set result should not be used.

## Sharpened Strategy

The strongest current result is the walk-forward LightGBM model-sort. The model is refit separately for each target year from 2015 through 2024 using only prior calendar years, then portfolios are formed on out-of-sample scores. The all-firm decile sort earns 22.49% annualized excess return with Sharpe 1.19 and FF5+UMD alpha 22.33% with t-stat 3.39. The most stable pre-specified variant is the tangible-heavy quintile sort: 13.35% annualized excess return, 10.77% volatility, Sharpe 1.24, max drawdown -4.84%, and FF5+UMD alpha 14.12% with t-stat 3.74. After a 10 bps one-way turnover haircut, the tangible-heavy variant retains 12.63% annualized return and Sharpe 1.17.
